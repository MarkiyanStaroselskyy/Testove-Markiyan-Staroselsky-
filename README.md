# Тестове завдання: Облік подій

Проста веб-сторінка для додавання та обліку подій з використанням HTML, CSS та vanilla JavaScript.

## Функціональність
- Форма для додавання події (назва + тип: click/lead/sale).
- Таблиця з усіма подіями (назва, тип, дата/час).
- Блок статистики: загальна кількість + по типах.
- Кнопка "Очистити всі події".
- Дані зберігаються в localStorage (не зникають при оновленні сторінки).

## Як запустити локально
1. Скачайте репозиторій або відкрийте папку з проектом.
2. Відкрийте файл `index.html` у браузері (Chrome, Firefox тощо).
3. Готово! Додавайте події, очищайте список.
